## Tech Stack

**Tech:** Node 18

## Run Locally

Clone the project

```bash
  git clone https://github.com/wahyunurarizky/tdc-fe.git
```

Go to the project directory

```bash
  cd tdc-fe
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```

open localhost:5173

## Run Use Docker Compose

```bash
  docker-compose up -d --build
```

open localhost:3000
